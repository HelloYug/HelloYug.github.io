Late Comers Management System - Web App

  • Tech Stack: Python, Flask, MySQL, SQLAlchemy, SMTP (Gmail), Matplotlib, dotenv, Bootstrap
  • Platform: Web Application (Browser-based)
  • Github URL: Project Link

A secure and modern web application designed for school PTEs to digitally manage late arrivals of students. Built with Flask and MySQL, it streamlines record-keeping, automates parent notifications, and provides visual analytics.

Features include:

  • Secure login system for PTEs with session-based authentication
  • Late arrival recording with reason and student tracking
  • Automatic email alerts to parents on each late entry
  • Export all late records to CSV files
  • Graphical analytics dashboard using Matplotlib

AI Technical Breakdown: Late Comers Web Portal

The web portal is built on the Flask micro-framework with SQLAlchemy ORM for robust database abstraction. It features secure session management and CSRF protection for administrative actions. The frontend uses Bootstrap 5 for high-fidelity responsiveness across mobile and desktop devices, integrating Matplotlib-generated SVGs for real-time data visualization.
