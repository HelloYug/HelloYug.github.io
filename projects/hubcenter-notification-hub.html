HubCenter – Serverless Notification Hub

  • Tech Stack: HTML5, CSS3, JavaScript (ES6+), Google Apps Script, Google Sheets, Netlify Edge Functions
  • Live Demo: hubcenter.netlify.app
  • Access Request: Request Access via Email

HubCenter is a zero-cost serverless notification hub that leverages Google Sheets as a secure database and Google Docs as a powerful CMS. It features a modern glassmorphism UI, secure rich-text parsing (HTML to Markdown pipeline), real-time caching, and intelligent category filtering.

Designed as a specialized module for static websites, HubCenter allows non-technical users to push professional updates, documents, and blog posts with ease. It is a robust alternative to expensive enterprise CMS platforms.

  • Zero-Cost Infrastructure – Powered purely by Google Workspace and static frontend files.
  • Flawless Rich Text Rendering – Custom parser converts Google Docs formatting to native design tokens.
  • Intelligent SEO – Dynamic meta-tag injection and Open Graph previews via Netlify Edge Functions.
  • Blazing Fast Performance – Runtime memory caching and localStorage for instant document loading.
  • Real-time Fuzzy Search – Powered by Fuse.js for instant filtering through all notifications.
  • Newsletter System – Fully serverless pipeline for subscription and automated branded campaigns.
  • Admin Preview Mode – Securely verify drafts before they go live with a handshake authentication.
  • Dynamic RSS Feed – Live ?action=rss endpoint for seamless integration with news readers.

HubCenter is an ideal solution for professional firms, educational institutions, and corporate portals that require high-performance dynamic content updates within a static-first architecture.

AI Technical Breakdown: HubCenter

HubCenter implements a serverless orchestration layer using Google Apps Script to interface with Google Workspace APIs. It features a custom rich-text AST (Abstract Syntax Tree) parser that translates Google Docs formatting into semantic HTML/Markdown. Edge caching via Netlify Edge Functions ensures sub-second delivery of dynamic content on a globally distributed CDN.
