DataSynapse Enterprise Backup Automation Suite

  • Tech Stack: PowerShell, Windows
  • Github URL: Project Link

DataSynapse Enterprise Backup Automation Suite is a PowerShell automation tool built specifically for BUSY Accounting Software. It creates a consolidated ZIP archive containing:

    1. The company data directory created by BUSY in the backend.
    2. The latest backup files generated by BUSY’s in-software backup service (e.g., DATA.BKP, DISK1.DB).

The script automatically detects the most recent backup among BUSY’s 7-day rotation and combines it with the live data into a timestamped ZIP folder. All original files and directories remain safe and unchanged.

Key Features:

  • Automatically detects the latest backup from BUSY’s 7-day rotation
  • Combines company data and latest backup files into a single ZIP
  • Generates readable logs and a per-company summary table
  • Safe and non-destructive to original BUSY data
  • Tested exclusively for BUSY Accounting Software
